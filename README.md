# Warcraft 3 Build Order Helper

A build order assistant for Warcraft 3: The Frozen Throne.

## Features
- **Dynamic Build Order Loader**: Load build orders directly from JSON files.
- **Configurable Timer**: Set a start delay for your build orders.
- **Icon Library**: Custom support for WC3 units and buildings.
- **Integrated Editor**: Easily modify or create new build orders.

## How to use
1. Open `Build Order Helper.html` in any modern web browser.
2. Select a build order from the list.
