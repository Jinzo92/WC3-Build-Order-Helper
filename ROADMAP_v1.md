# Roadmap Version 1.0 - WC3 Build Order Helper

Dies ist die Checkliste für den Release der Vollversion (V1.0):

## 1. Tutorial / "How to use"
- [ ] Hilfe-Button (?) in der Kopfzeile einbauen.
- [ ] Modal/Popup mit Erklärungen zu:
    - [ ] Hotkeys (z.B. Taste `S` zum Starten/Pausieren).
    - [ ] Editor-Nutzung (Items, Icons, Sounds).
    - [ ] Import/Export Funktionen.

## 2. Standard Build Orders (12 Stück)
- [ ] 12 professionelle Build Orders im Ordner `build-orders/` hinterlegen.
- [ ] Sicherstellen, dass diese beim Start automatisch geladen/gelistet werden.

## 3. UI & Design Feinschliff
- [ ] Design-Check: Farben, Abstände und Lesbarkeit optimieren.
- [ ] Eventuell dezente Animationen beim Fortschreiten der Time-Nodes.

## 4. User Feedback & Bugfixes
- [ ] Feedback von Testern einholen.
- [ ] Eventuelle Anzeigefehler oder Sound-Probleme beheben.

## 5. Finaler Release
- [ ] Update auf Version 1.0.0.
- [ ] Finale Push auf GitHub Pages.
